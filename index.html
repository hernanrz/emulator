Emu
Emu is a super simple server emulator written in PHP.

Why is it called an emulator?
Because I couldn't think of a better name it creates a server, but it will only serve whatever data you provide to it.


How is this useful at all?
I created this because some of the code in a different project makes several requests to a local API, so instead of installing another server I can just run Emu and test my code without much hassle.
